# 🎮 Number Guessing Game

> **A command-line number guessing game developed to practice Python fundamentals while introducing random number generation, game logic, loops, input validation, and problem-solving skills.**

---

# 📖 Overview

The **Number Guessing Game** is one of the foundational projects in this repository, designed to strengthen Python programming skills through an interactive command-line application.

The objective of the game is simple: the program randomly selects a number within a chosen difficulty level, and the player attempts to guess it. After each guess, the program provides feedback indicating whether the guess is too high or too low until the correct number is found.

Although simple in concept, this project introduces important programming concepts such as randomization, iterative problem solving, program flow, and user interaction. It also demonstrates how Python can be used to create engaging console-based applications.

# ⚙ Features

- Three difficulty levels
  - Easy (1–10)
  - Medium (1–50)
  - Hard (1–100)
- Random number generation
- High/Low hints after every guess
- Attempt counter
- Input validation
- Replay option
- Menu-driven command-line interface

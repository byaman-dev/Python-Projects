# 🔑 Password Generator

> **A command-line password generator developed to practice Python fundamentals while introducing randomization, string manipulation, input validation, and secure password generation principles.**

---

# 📖 Overview

The **Password Generator** is one of the foundational projects in this repository, designed to strengthen Python programming skills through a practical cybersecurity-related application.

Passwords remain one of the most important layers of digital security. This project demonstrates how Python can be used to generate customizable, random passwords while reinforcing programming concepts such as functions, loops, modules, and user input validation.

Although simple in implementation, the project introduces the idea of writing programs that solve real-world problems and lays the foundation for more advanced security-focused tools in later modules.

---

# 🎯 Objectives

This project was created to:

- Practice working with Python modules
- Learn random number and character generation
- Understand string manipulation
- Build reusable functions
- Validate user input
- Apply programming concepts to a practical cybersecurity tool
- Write clean and maintainable Python code

---

# ⚙ Features

- Generate passwords of custom length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Input validation
- Menu-driven command-line interface
- Generate multiple passwords without restarting the program

# 🧮 Calculator

> **A simple command-line calculator developed to practice Python fundamentals including user input, arithmetic operations, functions, and program flow.**

---

# 📖 Overview

The Calculator project is one of the first practical applications developed in this repository.

Although simple in functionality, it introduces many of the core programming concepts required for larger software projects. The program performs basic arithmetic calculations while demonstrating how user input can be processed, validated, and transformed into meaningful output.

Rather than being just a calculator, this project represents an important milestone in understanding structured programming.
